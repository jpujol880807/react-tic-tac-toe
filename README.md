# react-tic-tac-toe

A simple Tic\-Tac\-Toe game built with React.

Live demo: https://jpujol880807.github.io/react-tic-tac-toe

1. Installation
    1. Clone the repository: `git clone https://github.com/jpujol880807/react-tic-tac-toe.git`
    2. Enter the directory: `cd react-tic-tac-toe`
    3. Install dependencies: `npm install`
    4. Run in development: `npm start`
